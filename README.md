## Customer Segmentation using RFM Analysis

### Overview
This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis on a fashion retail sales dataset sourced from Kaggle.

### Tools Used
- Python
- Pandas
- Jupyter Notebook

### Dataset
- Source: Kaggle (Fashion Retail Sales Dataset)

### Key Business Questions Answered
- Who are the highest-value customers?
- Are customers becoming repeat buyers or one-time buyers?
- Which high-value customers are at risk of churn?
- Which customers should be avoided in paid targeting?

### Output
- RFM scores and customer segments
- Identification of high-value, churn-risk, and low-value customers
